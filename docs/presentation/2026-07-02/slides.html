The failure

The requirement became optional.

Prompt

"Do not send without approval." Good until context shifts.

Skill

Reusable procedure. Still soft influence, not durable state.

Handoff

Useful, but easy to lose across agents, retries, and restarts.

If it must survive tomorrow, it should not live only in a prompt.

The move

Promote obligations into workflow state.

Typed work

Agent and bash steps have inputs, outputs, fingerprints, and receipts.

Typed review

Review Queue cards come from schemas, not mystery approval buttons.

Worker continuation

The resident worker owns continuation after the initiating process exits.

Authoring surface

Small API. Ordinary Python.

from hermes_workflows import agent, ask, bash, goal, parallel, pipeline, workflow

Use agents for judgment. Use workflows for obligations: review gates, checks, side-effect boundaries, durable artifacts, and restartable execution.

Runtime model

Run records.
Worker continues.
Review resumes.

The CLI does not pretend to be a forever process. The worker leases queued commands and stops at typed human input.
